# Azure Networking Labs – Overview

This folder contains a collection of hands-on labs focused on **Azure Networking**.  
The purpose is to practice and understand core networking concepts by applying them in real lab scenarios using Azure Portal, CLI, and PowerShell.

---

## 📌 What You'll Find Here
Each lab in this folder is designed to:
- Explain a specific networking concept
- Apply it 
- Test different scenarios
- Highlight key takeaways and best practices

---

## 🧭 Topics Covered

### 1. Virtual Network (VNet)
- VNet fundamentals (Address Space, Subnets)
- Creating VNets (Portal & CLI)
- Network design basics

---

### 2. Network Security Groups (NSG)
- NSG concepts and rules
- Subnet vs NIC level security
- Real scenarios (Allow / Deny traffic)
- Application Security Groups (ASG)
- Service Tags & Augmented Rules

---

### 3. Connectivity & Peering
- VNet Peering
- VNet-to-VNet connections
- Comparing connectivity options

---

### 4. Private Access
- Service Endpoints
- Private Link & Private Endpoints
- Secure access to Azure services

---

### 5. Routing & Security
- Default and Custom Routes
- Azure Firewall
- Azure Bastion
- Basic DNS & Load Balancing concepts

---

## 🎯 Objective
- Build strong foundation in Azure Networking  
- Understand how cloud network security works  
- Practice real-world configurations  
- Prepare for **AZ-104 Certification**

---

## ⚙️ Tools & Technologies
- Azure Portal  
- Azure CLI  
- Azure PowerShell
